HTML Tags

These tags are the most widely used tags for making divisions and sections for html content mixed with css properties, these tags can make different designs and structures to a website.

div

div tag is a division and used to group HTML elements

section

section tag is defines a section in a document such as headers, footers, chapters and other section in the document

nav

nav tags is used to define as the set of navigation links either within the documents or to the other

span

span is used to color a part of a text
